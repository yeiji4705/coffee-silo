이미지 관리 방법 (관리자용)

1단계: GitHub 저장소에 `images` 폴더를 만드세요.

2단계: 각 탭별로 하위 폴더를 만드세요:
  • `images/tab1/` - 아크 12F
  • `images/tab2/` - 아크 4F
  • `images/tab3/` - 지재센 12F
  • ... (tab7까지)

3단계: 각 폴더에 이미지를 업로드하세요 (1.jpg, 2.jpg, 3.jpg ...)

4단계: 아래 HTML 코드에서 각 탭의 슬라이드를 수정하세요:

<div class="slide active">
  <img src="images/tab1/1.jpg" alt="아크 12F 사진 1">
</div>
<div class="slide">
  <img src="images/tab1/2.jpg" alt="아크 12F 사진 2">
</div>

💡 팁: 세로 이미지(9:16 비율)에 최적화되어 있습니다. 첫 번째 슬라이드에만 `active` 클래스를 추가하세요.
